# Markdown 快速入门指南

## 📌 1. 标题

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果：**
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

---

## 📝 2. 文本格式

### 2.1 基本样式
```
**加粗文本** 或 __加粗文本__
*斜体文本* 或 _斜体文本_
***加粗斜体*** 或 ___加粗斜体___
~~删除线文本~~
```

**效果：**
- **加粗文本**
- *斜体文本*
- ***加粗斜体***
- ~~删除线文本~~

### 2.2 段落和换行
```
这是第一段文字。
（空一行）
这是第二段文字。

这是第一行  
这是第二行（两个空格+回车）
这是第三行
```

## 📋 3. 列表

### 3.1 无序列表
```
- 项目一
- 项目二
  - 子项目一（缩进2空格）
  - 子项目二
* 也可以使用星号
+ 或者加号
```

**效果：**
- 项目一
- 项目二
  - 子项目一
  - 子项目二
* 也可以使用星号
+ 或者加号

### 3.2 有序列表
```
1. 第一项
2. 第二项
   1. 子项一（缩进3空格）
   2. 子项二
3. 第三项
```

**效果：**
1. 第一项
2. 第二项
   1. 子项一
   2. 子项二
3. 第三项

### 3.3 任务列表
```
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个任务
```

**效果：**
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个任务

## 🔗 4. 链接和图片

### 4.1 链接
```
[百度](https://www.baidu.com)
[带标题的链接](https://example.com "鼠标悬停文字")

直接链接：
<https://www.example.com>
```

**效果：**
[百度](https://www.baidu.com)  
<https://www.example.com>

### 4.2 图片
```
![图片描述](https://via.placeholder.com/150x100 "可选标题")

带链接的图片：
[![图片](图片地址)](链接地址)
```

## 📊 6. 表格

## 💎 10. 转义字符

```
\# 显示为普通#号
\* 显示为普通星号
\[ \] 显示为方括号
\( \) 显示为括号
\_ 显示为下划线
\\ 显示反斜杠
```

**效果：**
\# \* \[ \] \( \) \_ \\

## 💡 温馨提示

1. **先学常用功能**：掌握标题、列表、链接、代码块即可应对80%场景
2. **保持简洁**：Markdown的优势在于简洁易读
3. **多实践**：在GitHub、博客、笔记中多使用
4. **注意兼容性**：不同平台可能支持不同的扩展语法

> **记住**：Markdown不是排版工具，而是**写作工具**。专注于内容，而不是样式。
